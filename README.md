tp-tadp-2014-traits
===================


##Setup

####Install rvm

```bash 
   rvm install 2.1.1
```

####force login over bash (if you use other shells like zsh)

```bash 
    /bin/bash --login
```

####Set current vm version

```bash
    rvm use 2.1.1
```
####Install Rspec gem.

```bash 
    gem install rspec
```
